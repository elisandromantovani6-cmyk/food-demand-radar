# Story 1.5 — Integracao tRPC Frontend-Backend

**Status:** Done
**Epic:** EPIC-1-MVP

## Descricao
Como sistema, quero que todos os widgets e paginas consumam dados via tRPC em vez de dados hardcoded.

## Criterios de Aceitacao
- [x] HungerScoreGauge usa trpc.demand.getCityScore
- [x] HotNeighborhoods usa trpc.demand.getHotNeighborhoods
- [x] DemandTimeline usa trpc.demand.getTimeline
- [x] DemandHeatmap usa trpc.demand.getHeatmapData
- [x] AlertsPanel usa trpc.alerts.getActive + acknowledge mutation
- [x] CampaignsPage usa trpc.campaigns.getSuggested
- [x] TrendsPage usa trpc.trends.getFlavors
- [x] ExpansionPage usa trpc.expansion.getRanking
- [x] Loading skeletons em todos os componentes
- [x] Refetch automatico configurado por componente
